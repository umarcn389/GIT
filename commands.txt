git init
git add file_name(.)
git commit -m "Message"
git status-->to see the changes that has been done
git diff-->to specifically see what all changes that has been done.
git log-->logs of the commit that we have done
git checkout id--->it checks what changes have been done in that id, it moves the head to that id.
git branch-->it shows the branch and head status.
git checkout master--->attaches the head with the master branch
git checkout -b branch_name -->Switches to a new branch.
git merge branch_name--->be in master branch and then merge the new branches to master.
